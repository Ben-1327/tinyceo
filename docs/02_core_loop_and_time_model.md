# 02 Core Loop & Time Model

## 時間モデル（重要）
このゲームは “作業の横” で意味あるスピード感を出すため、**会社内時間を加速**する。

- Real Minute（現実の1分） = Company Minute × TIME_SCALE
- デフォルト `TIME_SCALE = 12`
- **2時間（現実）ごと**に CEOカードが 1枚発生  
  → 1サイクル（Cycle） = 2h real = 1 company day (24h)

### Active Session（時間が進む条件）
時間が進むのは「PC作業中」のみ（デフォルト）。
- `isSessionActive = true` のときだけ company time を進める
- 目安:
  - 画面がスリープしていない
  - ロックされていない
  - 最終入力（キーボード/マウス）から `idleThresholdMin` 未満（デフォルト 10分）
- オプション: 「アイドル中も進める」を ON にすると放置プレイも可能（ただし倒産しやすい）

この仕様により、寝ている間に破産しない。

## コアループ（1サイクル）
1. CEOカードが届く（Inboxに入る/通知される）
2. プレイヤーは 1枚処理（基本3択、条件付きで2〜4択になりうる）
3. 背景シミュレーションが回り、プロジェクトが進み、収支が動く
4. 次のサイクルでまたCEOカード

## Backlog（カード未処理）ルール
- Inbox 最大: `maxInbox = 3`
- 3枚溜まった状態でさらにサイクルが進むと、**自動的に「悪いデフォルト」が適用**される（=社長不在の悪影響）
  - TeamHealth 低下
  - TechDebt 上昇
  - Reputation 小低下
  - “火消し”系カードの出現率が上がる

※「詰んだ」より「困った→立て直す」を狙う。

## 進行スケールの目安
- 1日8時間作業（=4サイクル）すると、会社内では4日進む
- 1週間使うと会社内で約1ヶ月ぶん進む感覚
