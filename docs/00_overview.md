# 00 Overview

## このゲームが狙う体験
- 仕事中、画面の隅で会社が成長する（「ながら」）
- でも放置だけではなく、**2時間に1回**、社長として意思決定をする
- 実際の作業（Slack/Sheets/Terminal/AI）を **会社の成長に変換**する  
  ※内容は収集しない。アプリ種別やドメインなどの「カテゴリ」シグナルだけを利用。

## 主要な差別化ポイント
1. **ITベンチャー経営**（現実寄り：採用・制度・キャッシュ・評判）
2. **デスクトップ常駐**（作業の邪魔をしない）
3. **CEOカード（意思決定）中心**でゲーム性を担保
4. **仕組み化が進むほど放置耐性が上がる**（成長=楽になる）

## 重要なデザイン原則
- **Low-interruption**: プレイは 1回 60〜120秒が基本（通知も控えめ）
- **No-creepy**: ログはカテゴリのみ。中身を読まない。ローカル保存。
- **Not-punishing**: 寝ている間に破産しない。放置は“じわっと悪化”に留める。
